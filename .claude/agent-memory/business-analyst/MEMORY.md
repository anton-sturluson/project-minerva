# Business Analyst Agent Memory

## Project Structure
- Reports go to `hard-disk/reports/{TICKER}/` with all artifacts in same dir
- Reusable code in `src/minerva/` (models.py, formatting.py, report_generator.py, valuation.py)
- Package manager: `uv` (always use `uv run python` or `uv add`)
- Python 3.12, hatchling build backend, pydantic for data models

## 5-Phase Workflow
1. **Research**: Launch multiple search agents in parallel — one per discrete topic. Topics based on research dimensions (business model, industry, financials, risks, etc.) but adapted to the company. Not a fixed count — launch more if gaps emerge.
2. **Extract & Synthesize**: Triage sources from search manifests, extract with extract agents, build context.md, fill gaps with additional searches.
3. **Data Structuring**: Raw data → CSVs in `data/` subdir
4. **Analysis Script**: `analyze_{ticker}.py` reads CSVs, runs valuations, generates matplotlib PNGs
5. **Report**: `report.md` with 11 sections, embedded PNGs inline

See [research-patterns.md](./research-patterns.md) for detailed workflow, search queries, valuation steps, and 7 Powers guidance.

## Key Principles
- **LTM over NTM**: Default to trailing metrics; forward only when projections add value
- **7 Powers framework**: Mandatory competitive analysis (barrier + benefit for each power)
- **SBC as real cost**: Always subtract in DCF; track as % of revenue and market cap
- **Flexible chart guidance**: Plot every key 10-K/10-Q metric with a meaningful trend + any thesis-relevant metric
- **Three required valuation charts**: sensitivity heatmap, football field, peer comparison

## Sector-Specific Analysis Notes
- **Audio/Media Streaming**: Music royalty costs (~65-70%) create structural margin ceiling. Key metrics: ARPU, churn, free-to-paid conversion. FX drag matters for EUR-reporting companies.
- **Freemium models**: Track MAU growth, paid conversion rate, ARPU separately. Price increases flow at near-100% incremental gross margin.
- **EdTech / Consumer Learning**: Key metrics: DAU, MAU, DAU/MAU ratio, paid subs, ARPU, LTV/CAC. AI disruption risk is dominant bear case. SBC as % of revenue critical.
- **EdTech / DUOL specifics**: DUOL FY2025: $1.04B rev (+39%), 52.7M DAU, 12.2M paid subs, $360M FCF (35% margin), $1B net cash. Strategic pivot to user growth (100M DAU target by 2028) sacrificing near-term margins (~25% adj EBITDA in FY2026 vs 30% in FY2025). AI disruption is dominant bear case but company is AI-native. Key user metrics: track DAU, MAU, DAU/MAU ratio (39.6%), paid penetration (subs/MAU = 9.2%), ARPU by tier.
- **Dual-class share structures**: Flag governance risk; founders retain voting control regardless of dilution.
- **Crypto Exchanges**: Revenue extremely cyclical. Model bull/base/bear with different volume assumptions. Stablecoin revenue (~98% GM) is highest-quality stream. WACC 11-13%. GENIUS Act is key catalyst.
- **USDC Revenue Sharing (COIN/CRCL)**: Coinbase gets 100% on-platform, 50% off-platform reserve interest. Circle paid Coinbase $248M (2022), $691M (2023), $908M (2024), ~$1.1B (2025 est). Coinbase platform share: 5% (2022) -> 22% (Q1 2025). Contract renews Aug 2026 (3-year auto-renewing). GENIUS Act Section 4(a)(11) creates legal overhang on the arrangement (effective ~Jan 2027). CoinMetrics Issues 307/317 are best analytical sources for USDC economics.
- **Stablecoin Issuers / CRCL**: Revenue 96% from reserve interest — essentially a leveraged rate bet. Use NET revenue (after distribution costs) as economic revenue, not gross. FY2025: $2.75B gross, $1.08B net. IPO $31 (Jun 2025), ATH $299, now ~$63. WACC 12%. USDC $75.3B, 25% share. Zero-rate = unviable.

## minerva.valuation API Reference (Confirmed Through Use)
- **DCFAssumptions**: `base_revenue`, `base_fcf`, `revenue_growth_rates` (list), `fcf_margins` (list), `wacc`, `terminal_growth_rate`, `net_cash`, `shares_outstanding`, `sbc_annual`, `sbc_growth_rate`, `projection_years`
- **CompsAssumptions**: `ntm_revenue`, `ntm_ebitda`, `ntm_fcf`, `net_cash`, `shares_outstanding`, `ev_revenue_multiple` (single float), `ev_ebitda_multiple` (single float), `p_fcf_multiple` (single float)
- **run_reverse_dcf**: `current_price`, `shares_outstanding`, `net_cash`, `base_revenue`, `fcf_margin_trajectory` (list), `wacc`, `terminal_growth`, `projection_years`
- **SOTPSegment**: `name`, `revenue`, `revenue_pct`, `ev_revenue_multiple`, `implied_ev`, `notes`
- **DCFResult**: `.price_per_share` (pre-SBC), `.price_per_share_ex_sbc`, `.pv_total_fcf`, `.pv_terminal_value`
- **CompsResult**: `.ev_revenue_implied_price`, `.ev_ebitda_implied_price`, `.p_fcf_implied_price`
- **ReverseDCFResult**: `.implied_revenue_growth`, `.implied_year5_revenue` (named year5 even for longer projections)
- **dcf_sensitivity_matrix**: returns `list[list[float]]` of ex-SBC prices

## Common Gotchas
- `python` not found — always use `uv run python`
- pydantic must be added as dependency: `uv add pydantic`
- Q4 annual results often release in Feb of following year
- Market share % vary by source — always note methodology
- SBC differs between annual and TTM — be precise about time period
- Bash tool sometimes denied — write files via Write tool and compute manually if needed
- Always `os.makedirs(exist_ok=True)` for output dirs in Python scripts
- pandas/matplotlib/numpy must be added as deps (`uv add pandas matplotlib numpy`) before analysis scripts run
- CSV field count must be consistent across all rows — empty fields must still be comma-separated
- When search/extract agent skills are unavailable, conduct research directly with WebSearch/WebFetch
- Many financial sites (investing.com, macrotrends, gurufocus, fool.com) return 403 on WebFetch — use alternative sources
- StockDividendScreener.com: Good source for segment revenue breakdowns (transaction vs subscription)
- Coinbase Q4 2025 report released Feb 12, 2026: FY2025 rev $7.2B, net income $1.3B, FCF $2.4B
- Deribit acquisition ($2.9B) closed Aug 2025; Coinbase now has spot+futures+perpetuals+options
- Coinbase One subscribers reached ~1M by end 2025; $30/mo subscription
- COIN beta is 3.70 (5Y) — use blended WACC of 11-13% rather than raw CAPM output (which is unrealistically high at ~25%)
